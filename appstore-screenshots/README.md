## Upload Instructions

1. Go to App Store Connect: https://appstoreconnect.apple.com/apps/6763657217/distribution/ios/version/inflight
2. Navigate to **Previews and Screenshots**
3. Click on **iPhone** tab → **6.5" Display**
4. Drag and drop all `iphone-*.png` files
5. Click on **iPad** tab → **13" Display** (or 12.9")
6. Drag and drop all `ipad-*.png` files
7. Save your changes
