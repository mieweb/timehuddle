Security & Trust

Built secure
from the ground up.

🍪

httpOnly Cookies

Session tokens are stored in server-only cookies. JavaScript on the page can never read them — protecting against XSS attacks.

📱

Native Bearer Tokens

Mobile apps use a separate secure token flow — no cookie confusion in native WebViews.

🔑

OAuth + GitHub

Sign in with GitHub or email/password. Passwords are bcrypt-hashed — never stored in plain text.

🌐

OIDC Provider

Full OpenID Connect support — other trusted apps in the ecosystem can authenticate through Huddle.

🔐

API Tokens

Personal access tokens for programmatic access — scoped, revokable, and never stored in raw form.

🛡️

Role-Based Access

Admin and member roles per team. Only owners and admins can edit/delete clock events and manage members.
